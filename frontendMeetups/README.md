## ⚙️ Project Setup

### 📁 Installation

1. Clone the repository:

```bash
git clone https://github.com/Tushar13Kumar/frontend-meetups.git
```

2. Navigate to project folder:

```bash
cd frontend-meetups/frontendMeetups
```

3. Install dependencies:

```bash
npm install
```

---

### ▶️ Run Development Server

```bash
npm run dev
```

Project will run on:

```
http://localhost:5173
```
